# Corporate VPD Project

## Overview
This project implements Virtual Private Database concepts using PostgreSQL Row Level Security (RLS).

## Features
- Role Based Access Control
- Session-based Filtering using current_setting()
- Application-specific DB roles
- Row Level Security Enforcement
- Multi-role Testing

## Tech Stack
- PostgreSQL
- SQL
- Row Level Security

## Setup Instructions

1. Run 01_database.sql
2. Run 02_tables.sql
3. Run 03_roles_and_users.sql
4. Run 04_seed_data.sql
5. Run 05_app_roles.sql
6. Run 06_rls_policies.sql
